# Telecom Portal Automation Project

## 📌 Project Overview

This project demonstrates UI automation testing for a telecom customer portal using:

- Python
- Pytest
- Playwright

The project automates telecom-related workflows such as:

- Customer creation
- Tariff plan creation
---

# 🌐 Application Under Test

Demo Telecom Website:

https://demo.guru99.com/telecom/index.html

---

# 🎯 Project Objectives

The goal of this project is to practice:

- UI automation testing
- Playwright with Pytest integration
- Real-world telecom scenarios
- Assertions and validations
